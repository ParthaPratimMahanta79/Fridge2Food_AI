#  Fridge2Food AI

> Turn your available ingredients into delicious meals using AI.

---

##  Overview

Fridge2Food AI is a simple and smart AI-powered web app that helps users cook meals using the ingredients they already have. Just enter what’s in your kitchen, and the AI will generate a quick and easy recipe.

---

## 💡 Problem

Many people struggle with:
- Not knowing what to cook
- Limited ingredients
- Wasting food

---

## Solution

Fridge2Food AI solves this by:
- Taking available ingredients as input
- Generating a recipe instantly
- Providing simple, beginner-friendly steps

---

##  Features

-  Enter ingredients you have
-  AI-generated recipes
-  Simple step-by-step instructions
-  Quick and beginner-friendly cooking

---

##  Tech Stack

- **Frontend:** HTML, CSS, JavaScript  
- **Backend:** Node.js, Express.js  
- **AI:** OpenAI API
